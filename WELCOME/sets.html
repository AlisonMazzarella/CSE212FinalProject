Sets

Introduction

Sets are used to store multiple items in a single variable. They are a collection which is unordered, unchangeable, and unindexed. Because sets are unordered items can appear in a different order every time you use them, and therefore cannot be referred to by index. Sets do not allow for duplicate values. While unchangeable, you can alter a set by removing items and adding new ones. Sets are identifiable by their use of curly brackets "{ }".

Hashing

Set operations are optimal when they perform at O(1) time. In order to keep this time when adding or removing items we use hashing.
