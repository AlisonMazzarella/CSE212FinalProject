Programming with Data Structures

Introduction

Data structures are the fundamental constructs around which you build your programs. Each data structure provides a particular way of organizing data so it can be accessed efficiently, depending on your use case.

The specific data structures we will be going over in this tutorial are sets, stacks, and trees. You will find in each tab on the navigation bar an introduction to each topic, examples, and practice problems to challenge yourself.
